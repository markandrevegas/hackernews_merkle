# Hackernews Assignment | Nuxt Minimal Starter

There are no plugins, ui frameworks, or css frameworks installed. Prettier is part of my normal workflow as i like my work to be easy to read (for myself).

localStorage was used to cache the initial response from [text](https://hacker-news.firebaseio.com/v0/topstories.json)


## Setup

Make sure to install dependencies:

```bash
# yarn
yarn install

```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# yarn
yarn run dev

```

## Production

Build the application for production:

```bash
# yarn
yarn run build

```

Locally preview production build:

```bash
# yarn
yarn run preview

```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
